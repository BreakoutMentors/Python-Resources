# Mouse Inputs

## Mouse Location
```python
print(mouse.x)
print(mouse.y)
``` 


## Mouse Clicked
```python
def mouseClicked():
  print "yes"
``` 

Or:
```python
print(mouse.pressed)
```
 
## Key Pressed
```python
def keyPressed():
  print(keyboard.key)
```
 
## Other things to know

Can also get the previous x or y for when the mouse moves:
```python
print(mouse.px)
print(mouse.py)
```
